## Overview

This project analyzes over 32,000 job postings from the data industry using Power BI. The dashboard provides insights into hiring demand, salary trends, and compensation patterns across roles such as Data Analyst, Data Scientist, Data Engineer, and Business Analyst.

## Business Questions

* Which data-related roles have the highest salaries?
* How has hiring demand changed over time?
* What are the median yearly and hourly salaries across different professions?
* Which roles are most frequently posted by employers?

## Tools Used

* Power BI
* Power Query
* DAX
* Excel

## Dashboard Features

* KPI Cards for job count and salary metrics
* Salary comparison by role
* Hiring trends over time
* Interactive filtering by job title
* Custom Salary Star Rating measure using DAX

## Key Insights

* Data Engineer and Senior Data Scientist roles showed the highest salary ranges.
* Data Analyst positions represented a significant share of total job postings.
* Salary levels varied considerably across different data professions.
